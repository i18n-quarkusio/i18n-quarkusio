リアクティブ入門

Quarkusでリアクティブアプリケーションを作成する方法を学習し、Quarkusによって提供される様々なリアクティブ機能を学びます。 このガイドは以下をカバーします：

  • Quarkusエンジンと、それがどのようにリアクティブを可能にしているかの概要

  • 概説Mutiny - Quarkusによって使用されているリアクティブプログラミングライブラリ

  • リアクティブアプリケーションのクイック起動

  • JAX-RSエンドポイントの作成（非同期、ストリーム…）

  • リアクティブデータベースアクセス

  • その他のリアクティブAPIの使用

ツールの使用

Quarkus には開発者がライブリロードからKubernetesへのアプリケーションのデプロイまで、ツールチェインが付属しています。 このガイドでは、以下について説明します：

  • Mavenをビルドツールとして使用する方法

  • Gradleをビルドツールとして使用する方法

  • ツールチェインにネイティブCLIを使用する方法（近日公開）

  • 新しいプロジェクトをスキャフォールド (骨組を作成) する方法

  • エクステンションの扱い方

  • ライブリロードの有効化

  • IDEでアプリケーションを開発する方法

  • IDEでアプリケーションをコンパイルする方法
