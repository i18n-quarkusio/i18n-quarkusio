quarkus 2.0 登場!

Quarkus 2.0は、新しい開発者ツールと改善されたパフォーマンスにより、JavaをKubernetesネイティブ開発に適したフレームワークにするという使命を継続します。

2.0 の詳細

A virtual tour of Java User Groups creating a unique hands-on experience with access to Quarkus experts designed to help you get started with Java in a Kubernetes world.

Learn more about our World Tour stops.

コンテナファースト

QuarkusはアプリケーションをGraalVMとHotSpot向けに最適化します。驚くべき高速起動と、信じられないほどの低RSSメモリ（ヒープサイズだけではありません！）により、Kubernetesなどのコンテナオーケストレーションプラットフォームでほぼ瞬時にスケールアップし、高密度のメモリ使用率を実現します。コンパイル時ブートと呼ばれる手法を使用します。詳細

$ ./my-native-java-rest-app
Quarkus started in 0.008s

命令型とリアクティブ型の統合

アプリケーション開発時に、慣れ親しんだ命令型コードと、グリアクティブ型コードを組み合わせることが出来ます。 詳細.

命令型
@Inject
SayService say;

@GET
@Produces(MediaType.TEXT_PLAIN)
public String hello() {
    return say.hello();
}
リアクティブ型
@Channel("kafka") Multi<String> events;

@GET
@Produces(MediaType.SERVER_SENT_EVENTS)
public Multi<String> events() {
    return events;
}

開発者満足

開発者満足度に最適化された凝集度の高いプラットフォーム:

  • 統一された設定
  • ゼロコンフィグ、瞬く間のライブ・リロード
  • 80%の一般的な使用法の為に合理化されたコード、20%の為の柔軟性
  • 容易なネイティブ実行可能ファイル生成
