Infinispanは、アプリケーションに直接埋め込むことができる拡張性の高いインメモリーデータストアです。

Infinispanプロジェクトの詳細については、Infinispanの ドキュメント をご覧ください。

この技術は、previewと考えられています。

とりうるエクステンションステータスの完全なリストについては、 FAQの項目を 参照してください。

Adding the Infinispan Embedded Extension

After you set up your Quarkus project, run the following command from the base directory:

./mvnw quarkus:add-extension -Dextensions="infinispan-embedded"

このコマンドは、以下の依存関係を pom.xml に追加します:

<dependency>
    <groupId>io.quarkus</groupId>
    <artifactId>quarkus-infinispan-embedded</artifactId>
</dependency>

機能サポート

Infinispan Embeddedエクステンションは、クラスター化キャッシュ、オフヒープメモリー、データ永続化、トランザクションを含むコアキャッシング機能を提供します。

Infinispan Embeddedエクステンションは、現在のところインデックス機能をサポートしていません。

トランザクション

Infinispan キャッシュを特定の TransactionManagerLookup でトランザクション処理用に設定するには、以下のようにします。

<local-cache name="quarkus-transaction">
   <transaction transaction-manager-lookup="org.infinispan.transaction.lookup.JBossStandaloneJTAManagerLookup"/>
</local-cache>

詳細については、Quarkus トランザクションガイド を参照してください。

ネイティブの制限

ネイティブモードでInfinispanを実行する場合、いくつかの制限が適用されます:

  • JMX管理には対応していません。 + GraalVMは、ネイティブVMインターフェースを許可していません。

  • UDP / Multicast はサポートされていません。 + JGroupsでは、 TCP トランスポートと非UDPベースのメンバーシッププロトコルを使用する必要があります。

インジェクション(CDI)

Infinispan Embeddedエクステンションはインジェクション機能を提供するので、手動でキャッシュを設定して起動する必要はありません。

追加のインジェクション機能は、将来のバージョンで利用可能になる予定です。
EmbeddedCacheManager

キャッシュを設定したり取得したりするためのメインのエントリーポイントです。

設定リファレンス

JMX要素はネイティブ実行時には無効化されるので、JVMとネイティブモードで同じ設定ファイルを使用することができます。
